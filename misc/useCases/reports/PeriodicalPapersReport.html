2) PLANNING WORKFLOW
Name Maintain Periodical Papers Reference
What should be done Workflow applied to create linked dataset of Periodical Paper References of the UNICENTRO's researchers, in an automatized way.
Precondition 1. Availability of the data (sql dump or what is there) 2. Running system with installed Ubuntu and all necessary tools (i.e. Sparqlify, Virtuoso, virt-load and all the scripts)
Postcondition PeriodicalPapers Graph is created and maintained.
