1) PROJECT IDENTIFICATION 
1.1 - Name: QualisBrasil Project
1.2 - Homepage: http://lod.unicentro.br/QualisBrasil/
1.3 - Creator: Sandro Rautenberg
1.4 - Goal: Offering Qualis Periodical Index according to the principles Linked Open Data
1.5 - Description: QualisBrasil is part of http://lod.unicentro.br endpoint. It aims to aid researchers to collect Linked Open Data about Qualis Index in bibliometrics or scientometrics studies. Available data: Periodical ISSN, Periodical Name, Knowledge Field, Qualis Index, and Year.

2.1.2.1 - step_extracting_qualisbrasil2005_from_legacy_system

Contributor: Sandro Rautenberg

Stage: Extraction

Performed at: 2014-11-06

Input Datasets:

dataset_lodprojectfinal20141106_1325_sql
 (File format: SQL Format)
 (Location: D:\LODPROJECTS\lodUnicentro\qualisBrasil\Datasets\Commons)
 (License: No license defined)

Output Dataset: 
dataset_qualisbrasil2005_csv
 (File format: CSV Format)
 (Location: D:\LODPROJECTS\lodUnicentro\qualisBrasil\Datasets\2005)
 (License: No license defined)

Tool: MySQL

Parameters and parameter's values:

parameter_year : 2005


Short Description: Restore the database in a mysql prompt (see hasInputDataset). Run the sql query (see hasScript). Save the output dataset (see hasOutputDataset).


Script: select
  concat('Periodico_',jo.issnJournal,'_Area_',sa.idCNPq,'_Ano_',josaqu.yearIndex,'_Qualis_',qu.nameQualis) AS Avaliacao,
  concat('Periodico_',jo.issnJournal) AS Periodico,
  jo.issnJournal AS temISSNPeriodico,
  jo.nameJournal AS temNomePeriodico,
  concat('Area_',sa.idCNPq) AS Area,
  sa.idCNPq AS temIdArea,
  sa.nameSubArea AS temNomeArea,
  concat('Ano_',josaqu.yearIndex) AS Ano,
  josaqu.yearIndex AS temAno,
  concat('Qualis_',qu.nameQualis) AS Qualis,
  qu.nameQualis AS temQualis
from
  journals jo,
  journalssubareasqualis josaqu,
  qualis qu,
  subareas sa
where (
  jo.id = josaqu.fk_journals and
  josaqu.fk_qualis = qu.id and
  josaqu.fk_subareas = sa.id and
  josaqu.yearIndex = 2005
) order by josaqu.yearIndex, jo.nameJournal;

Script: select
  concat('Periodico_',jo.issnJournal,'_Area_',sa.idCNPq,'_Ano_',josaqu.yearIndex,'_Qualis_',qu.nameQualis) AS Avaliacao,
  concat('Periodico_',jo.issnJournal) AS Periodico,
  jo.issnJournal AS temISSNPeriodico,
  jo.nameJournal AS temNomePeriodico,
  concat('Area_',sa.idCNPq) AS Area,
  sa.idCNPq AS temIdArea,
  sa.nameSubArea AS temNomeArea,
  concat('Ano_',josaqu.yearIndex) AS Ano,
  josaqu.yearIndex AS temAno,
  concat('Qualis_',qu.nameQualis) AS Qualis,
  qu.nameQualis AS temQualis
from
  journals jo,
  journalssubareasqualis josaqu,
  qualis qu,
  subareas sa
where (
  jo.id = josaqu.fk_journals and
  josaqu.fk_qualis = qu.id and
  josaqu.fk_subareas = sa.id and
  josaqu.yearIndex = 2013
) order by josaqu.yearIndex, jo.nameJournal;
